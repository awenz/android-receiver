# Android Receiver

Listen for broadcast messages from [android-notifer][]. Display them 
using `notify-send`.

## Installation

~~~
$ make
$ sudo make install
~~~

A PKGBUILD is present for Arch users.

## Usage

Enable "Send with UDP" in android-notifier.

~~~
$ android-receiver
~~~

[android-notifer]: http://code.google.com/p/android-notifier/
